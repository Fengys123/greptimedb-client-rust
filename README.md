# GreptimeDB Rust Client

A Rust client for GreptimeDB gRPC protocol.

## Status

- [x] gRPC inserts
- [ ] gRPC stream inserts
- [x] gRPC deletes
- [ ] Arrow Flight API
  - [ ] DDL
  - [ ] SQL Query
  - [ ] PromQL Query
- [ ] Prometheus Gateway API
